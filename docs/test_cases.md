# 圆小付系统测试用例

本文档记录“圆小付”支付系统 mock 项目的核心测试用例。用例围绕登录鉴权、账户概览、转账、个人资料、基金购买、交易流水和前端入口展开，重点覆盖支付类系统常见的金额边界、重复提交、余额变化、未授权访问和响应结构稳定性。

## 1. 测试范围

| 模块 | 测试重点 |
| --- | --- |
| 登录 | 正常登录、密码错误、必填参数、Token 返回 |
| 前端入口 | 未登录只展示登录页，登录后进入业务工作台 |
| 账户概览 | 登录后查询账号、昵称、手机号、余额 |
| 转账 | 金额边界、余额扣减、重复请求号、未登录访问 |
| 个人资料 | 昵称修改、手机号格式、未登录拦截 |
| 基金 | 基金列表、开放基金购买、关闭基金拦截、起购金额 |
| 交易流水 | 转账入流水、基金购买入流水、未登录拦截 |
| 静态资源 | 首页、CSS、JavaScript 可访问 |

## 3. 测试数据

| 数据项 | 值 | 说明 |
| --- | --- | --- |

| 关闭基金 | `FUND002` | 不可购买 |
